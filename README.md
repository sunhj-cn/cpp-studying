### 递增递减运算符 a++ ;++a
   ```
   int a = 20; 
   int b = 20;
   cout << "a++ = " << a++ << ";  ++b =" << b++ << endl;
   cout << "a = " << a << ";  b =" << b << endl; 
   ```
   输出结果：
   <a++ = 20; ++b = 21
    a = 21;  b = 21>
